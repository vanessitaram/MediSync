# MediSync
Atención clínica centralizada para el ingreso del paciente, apoyado para consulta prediagnóstica con IA y smartwatch.

## Funcionalidades

1. [] Registro de datos del paciente
2. [] Captura de síntomas.
3. [] Chat del paciente con la IA
4. [] Generacion del prediagnostico automatico
5. [] Notificacion al medico cuando el paciente esta en espera
6. [] Visualizacion del chat y sintomas por el medico
7. [] Chat del medico con la IA para contrastar el prediagnostico
8. [] Registro de observaciones medicas
9. [] Cierre de consulta con diagnostico final
10. [] Expediente digital del flujo completo
11. [] Medicion de tiempos de espera y consulta
12. [] Control de acceso por roles
13. [] Cifrado y proteccion de datos
14. [] BAckend con API REST
15. [] Base de datos SQL
16. [] Interfaz web para paciente, medico y recepcion
